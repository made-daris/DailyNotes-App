# ManaNote

(2023/08/07  アプリ名が[ManaNote]に変更となりました。)

日常的なメモの保存ができるアプリです。  
ログイン機能が搭載されているので安心してお使いいただけます。
 
# イメージ(これより先はPC版画面での説明となります)
 
<img width="1440" alt="編集後" src="https://github.com/made-daris/DailyNotes-App/assets/140872620/5ce44014-464a-468a-a24f-8b51b2b581b0">
 
# 特徴
 
それぞれのメモにタグを割り当てる事で、タグごとの管理が可能です。  
シンプルであるが故に簡単に操作できます。
 
# ご利用手順
 
<a href="https://mananote.net">こちらのWEBサイト</a>にてログイン(新規ご利用の方は右上のタブから[Register]を選択し登録)されるとご利用いただけます。

# 使い方

1. まず初めに[新規メモ作成]フォームにメモを入力します。
<img width="1440" alt="保存前" src="https://github.com/made-daris/DailyNotes-App/assets/140872620/4c3834ec-f985-4957-893d-35153cff6475">

2. 任意のタグ名を決めて保存ボタンを押すと[メモ一覧]に保存されます。
<img width="1440" alt="保存後" src="https://github.com/made-daris/DailyNotes-App/assets/140872620/a89831ae-f4b6-4cde-9932-bd521ccb4b59">

<img width="1440" alt="新規予定" src="https://github.com/made-daris/DailyNotes-App/assets/140872620/1503091a-8861-4580-abe9-504c44feec6e">

3. [タグ一覧]からタグを選択することで同じタグのメモのみを確認できます。
<img width="1440" alt="タグで絞り込み" src="https://github.com/made-daris/DailyNotes-App/assets/140872620/746cd905-a356-4bba-9943-158c33881e36">

4. 一度保存したメモの内容を編集したい場合は、該当するメモを選択後[メモ編集]フォームで内容を変更し、更新ボタンを押してください。
<img width="1440" alt="編集前" src="https://github.com/made-daris/DailyNotes-App/assets/140872620/fca0758f-f5d0-4f64-b814-3b3ed5ab0af2">

<img width="1440" alt="編集後" src="https://github.com/made-daris/DailyNotes-App/assets/140872620/5ce44014-464a-468a-a24f-8b51b2b581b0">

5. メモを削除したい場合は、該当するメモを選択後画面右上のゴミ箱ボタンを押してください。
<img width="1440" alt="削除" src="https://github.com/made-daris/DailyNotes-App/assets/140872620/222e66c3-9743-4f29-8b76-596310cfbb22">

<img width="1440" alt="削除後" src="https://github.com/made-daris/DailyNotes-App/assets/140872620/10bf93f8-d170-4c71-babd-967ceabb46b3">

 
# 注意
 
一度削除されたメモは復元ができませんのでご注意ください。
